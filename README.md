## Getting Started

### Prerequisites

- Node.js (14+ recommended)
- npm or yarn

### Install

1. Clone the repo:

   git clone https://github.com/adil-dev96/cards-React.git
   cd cards-React

2. Install dependencies:

   npm install
   # or
   yarn install

### Run

Start the development server:

   npm start
   # or
   yarn start

Open http://localhost:3000 in your browser.

Build for production:

   npm run build
   # or
   yarn build
